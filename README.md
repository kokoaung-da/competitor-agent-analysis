# Competitor’s Agent Analysis (Excel & Python)

- Collected agent shop data from **Wave Money’s website** by extracting and converting **PDF files** into structured **Excel datasets** using **Python**.
- Performed **data cleaning and quality checks** to ensure accuracy and completeness.
- Conducted **exploratory data analysis** to evaluate **agent distribution by region and township**, including detailed insights on agent names, shop names, and addresses.
- Designed an **interactive Excel dashboard** to visualize regional and township-level agent distribution, enabling easier competitor tracking and decision-making.

**Key Findings & Insights**

- **Nationwide Presence:**
    
    A total of **59,533 Wave Money agents** are distributed across **297 townships**, reflecting strong national coverage and accessibility of Wave Money services.
    
- **Regional Concentration:**
    
    The **Yangon Region** dominates with **15,023 agents (25.2%)**, showing the highest agent density — likely driven by urban population, higher transaction volumes, and digital readiness.
    
    Other leading regions include **Mandalay (13.9%)** and **Ayeyarwady (11.4%)**, indicating Wave Money’s strategic expansion into populous and economically active areas.
    
- **Rural Penetration Opportunity:**
    
    Regions such as **Chin, Kayah, and Shan (East)** have less than **1% agent representation**, suggesting significant **growth potential for agent network expansion** in underserved and rural regions.
    
- **Top Townships by Agent Count:**
    - **Bago (1,493 agents)** leads township-level distribution, followed by **Dagon Myothit (North)** and **Pathein**.
    - These townships act as **high-volume transaction hubs** and could serve as **logistics or marketing anchor points** for future regional growth.
